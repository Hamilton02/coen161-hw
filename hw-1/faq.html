When do you use an <ol> compared to <ul>

You use an <ol> when you just want to have bullet points and a <ul> when you want to keep track of the number of items.

What does the alt attribute of an <img> do?

The alt attribute will describe the image in case the picture won't load or the user is using a screen reader

How do you use an <a>

Wrap an element with <a> tags and use href in the leading one to link to another page

With correct semantics, are you allowed to have more than 1 <header> element per HTML page?

There is nothing stopping someone from having multiple <header> tags in their html documents

How do you indicate that a <video> can have multiple different sources?

You can indicate there is multiple sources by using using <source> inside of the <video> tags
