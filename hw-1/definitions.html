Defintions

<p>
This is the paragraph element that should be used for paragraph text and defaults to a size of 1rem or 16px.
<Aside>
This can be used to make a side column apart from the main body and will help screen readers decipher what they're seeing.
<body>
The body tag is used to tell the browser what goes on the page
<hr>
The hr tag or horizontal row tag is used to make a line horizontally between element
<button>
The button element creates a button that can be clicked and using the <a> tag can be used to link to something
<script>
The script tag can be used to enter javascript into a page without making a seperate javascript file
